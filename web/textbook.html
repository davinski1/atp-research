数学1

数学の基礎

数学の基礎を準備します。一階論理から始まって、集合・クラス・対・写像・代数の初歩を扱います。 本教科書は「数学を記号列の変換ゲームとして扱う」という立場をとり、 機械による定理証明（ATP）を念頭に置いた記述を行います。

本教科書の方針

ルールは「機械化できれば完全に厳密でなくても良い」とします。 表現はできるだけ大学数学で使われるものを採用し、 意味や自然言語に依存しない、機械が処理できるルールを優先します。

1-a 基本論理記号

a1. 一階言語の記号

一階言語で共通に用いる記号を導入します。

定義: 変数

変数はラテン文字の小文字・大文字 (a–z, A–Z)、カリグラフィー体 (𝒜–𝒵)、 フラクトゥール体 (𝔞–𝔷, 𝔄–𝔷)、ギリシャ文字 α、 および添字付き変数 x₀, x₁ などを用います。

定義: 命題記号と等号
種類記号意味
零項（定数）⊤, ⊥真, 偽
単項¬否定 (NOT)
二項結合子∧論理積 (AND)
∨論理和 (OR)
→, ⇒含意 (IMPLIES)
↔, ⇔同値 (IFF)
等号=等しい
定義: 量化子
∀x. P    — 全称量化「すべての x について P」
∃x. P    — 存在量化「ある x が存在して P」

複数変数の量化は次のように略記します:

∀x₁, x₂, ..., xₙ. P  ≡  ∀x₁. ∀x₂. ... ∀xₙ. P
定義: 二項述語の性質
記号名称定義
I反射律∀x. (x 𝖕 x)
T推移律∀x,y,z. (x 𝖕 y ∧ y 𝖕 z → x 𝖕 z)
X対称律∀x,y. (x 𝖕 y → y 𝖕 x)
定理 a1.1
= は同値関係である（= : I T X）

等号は反射律・推移律・対称律をすべて満たす。

a2. 言語の拡張

一階言語の表現力を拡張する仕組みを導入します。

定義: 代入

式 P 中の変数 x を項 T で置換する操作を次のように書きます:

P(x↦T)
定義: 擬単項関数 (fn)

記法 [x | X] は fn x(X) を表します。 •（ブレット）は第一変数の略記です。

id = [x | x]   （恒等関数）
定義: クラス (cls)

記法 {x | P} は cls x(P) を表します。

[T ∈ {x | P}]  ≡  P(x↦T)

クラスの等号は外延性により定義されます。

定義: 一意性
∃!x. P  ≡  ∃x. (P ∧ ∀y. (P(x↦y) → y = x))

a3. 集合とクラス

定義: 集合

集合はクラスの一種とみなします。変換:

X ≃ {x | x ∈ X}
定義: 部分集合
X ⊂ Y  ≡  ∀x ∈ X. x ∈ Y
X ⊊ Y  ≡  X ⊂ Y ∧ X ≠ Y
定理 a3.1
⊂ は反射律と推移律を満たす（⊂ : I T）
定理 a3.2
X = Y ⟺ X ⊂ Y ∧ Y ⊂ X

集合の等号は相互包含で特徴づけられる。

定義: 有限集合の生成

演算子 set_n により n 個の要素から集合を生成します:

{X₁, ..., Xₙ} = set_n(X₁, ..., Xₙ)

1-b 集合の構成

b1. 具体的な集合とクラスの生成

定義: 全体クラスと空集合
𝕍 = {x | ⊤}   （全体クラス / Universe）
∅ = {x | ⊥}   （空集合 / Empty set）
定義: 集合演算
演算記号定義
和集合X ∪ Y{x | x ∈ X ∨ x ∈ Y}
共通部分X ∩ Y{x | x ∈ X ∧ x ∈ Y}
差集合X ∖ Y{x | x ∈ X ∧ x ∉ Y}
定義: 合併・共通部分（集合族）
⋃𝒜 = {x | ∃A ∈ 𝒜. x ∈ A}
⋂𝒜 = {x | ∀A ∈ 𝒜. x ∈ A}

注意: ⋂∅ = 𝕍

定義: ラッセルのパラドクス

𝕍₀ = {x | x ∉ x} とおくと、 𝕍₀ ∈ 𝕍₀ と仮定しても 𝕍₀ ∉ 𝕍₀ と仮定しても矛盾が生じる。 これは 𝕍₀ が集合ではなく真クラス (proper class) であることを示す。

公理: 存在公理
ax_s(C) : ∀X. ∃(X ∩ C)

クラス C が集合を生成する条件。

b2. 自然数

フォン・ノイマンの構成により自然数を定義します。

定義: 自然数の構成
0 = ∅
n + 1 = n ∪ {n}

具体的に: 0 = ∅, 1 = {∅}, 2 = {∅, {∅}}, 3 = {∅, {∅}, {∅, {∅}}}, ...

定義: 帰納的集合と ℳ
Ind = {X | ∅ ∈ X ∧ ∀n ∈ X. (n+1) ∈ X}
ℳ = ⋂ Ind

ℳ は帰納的集合の全体の共通部分で、自然数全体の集合です。

定理 b2.1
m, n ∈ ℳ ⟹ (m ∈ n ⟺ m ⊂ n)

自然数の間では、所属関係と部分集合関係が一致する。

b3. 自然数の演算

加法・乗法を帰納的に定義し、ℳ 上の閉性を示します。

定義: 加法と乗法
m + 0 = m,    m + (n+1) = (m + n) + 1
m × 0 = 0,    m × (n+1) = m × n + m

1-c 対と関数

c1. 対と直積

定義: 順序対
⟨x, y⟩ ≈ pr(x, y)

成分の取り出し:

◁⟨x, y⟩ = x   （左成分）
▷⟨x, y⟩ = y   （右成分）
定理 c1.1
⟨x₀, y₀⟩ = ⟨x₁, y₁⟩ ⟺ x₀ = x₁ ∧ y₀ = y₁
定義: 関係と直積
Rel ≃ {R | ∀x ∈ R. x = ⟨◁x, ▷x⟩}
X × Y = {⟨x, y⟩ | x ∈ X ∧ y ∈ Y}
定義: 定義域と像
dom(R) = {x | ∃y. ⟨x, y⟩ ∈ R}
im(R) = {y | ∃x. ⟨x, y⟩ ∈ R}

c2. 写像

定義: 写像
X → Y = {f ⊂ X × Y | X ⊆ dom!(f)}

ここで dom!(f) = {x | ∃!y. ⟨x,y⟩ ∈ f}

定義: 単射・全射・全単射
種類記号条件
単射 (Injective)I異なる入力に異なる出力
全射 (Surjective)Sim(f) = Y
全単射 (Bijective)IS単射かつ全射
定義: 恒等写像
id_X = {⟨x, x⟩ | x ∈ X}

c3. 濃度

定義: 濃度の比較
X ≂= Y  ⟺  X →IS Y ≠ ∅   （等濃度）
X ≂≤ Y  ⟺  X →I Y ≠ ∅   （濃度 ≤）
定義: 有限集合
𝕍+∞ ≃ {X | ∃n ∈ ℳ. X ≂= n}   （有限集合の全体）

1-d 代数の初歩

d1. 二項演算の性質

定義: 二項演算の基本性質
記号性質定義
閉閉性 (Closure)∀x,y ∈ X. x f y ∈ X
C可換律 (Commutativity)∀x,y ∈ X. x f y = y f x
A結合律 (Associativity)∀x,y,z ∈ X. x f (y f z) = (x f y) f z
J冪等律 (Idempotence)∀x ∈ X. x f x = x
定義: 単位元
unit(X, f) = {e | ∀x ∈ X. e f x = x = x f e}

d2. 自然数と整数

定義: ℕ と ℤ
ℕ = ℳ ∖ {0}   （正の自然数）
ℤ = ℳ ∪ (−ℕ)   （整数）

d3. 冪集合

定義: 冪集合
℘(X) = {A | A ⊂ X}
定理 d3.1 (Cantor)
X < ℘(X)

任意の集合 X について、その冪集合は X より真に大きい濃度を持つ（カントールの定理）。

定義: 有限部分集合
℘∞(X) = ℘(X) ∩ 𝕍+∞

1-e 同値関係と有理数

定義: 同値関係

集合 X 上の二項関係 ∼ が次の3条件を満たすとき、同値関係という:

I（反射律）: ∀x ∈ X. x ∼ x
X（対称律）: ∀x,y ∈ X. (x ∼ y → y ∼ x)
T（推移律）: ∀x,y,z ∈ X. (x ∼ y ∧ y ∼ z → x ∼ z)
定義: 同値類と商集合
[x]∼ = {y ∈ X | x ∼ y}   （x の同値類）
X / ∼  = {[x]∼ | x ∈ X}   （商集合）
定義: 有理数 ℚ

整数の対の集合に同値関係を入れて有理数を構成します:

ℤ × ℕ 上の同値関係: ⟨a, b⟩ ∼ ⟨c, d⟩ ⟺ a × d = c × b
ℚ = (ℤ × ℕ) / ∼
